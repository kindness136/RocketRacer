操作方法
PC　矢印左：左へ移動　矢印右：右へ移動

Unity上で初めて製作したゲームになります。
ロケットを操作し、向かってくる岩をかわしながらゴールのタイムを競うゲームです。１プレイ１分前後で終わるシンプルなゲーム性を目指して製作しました。
